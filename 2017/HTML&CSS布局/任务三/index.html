FE-Developer

我们来自四川省成都市西华大学微力实验室，也来自不同的专业、 不同的年级，源于对前端技术的热爱与追求，组成FE developer,希望能在百度前端技术学院，学习到更多前端技术，就在上个星期，硅谷最伟大的偏执狂离开了我们——英特尔前CEO、著名管理大师安迪·格鲁夫去世，享年79岁。可能正抱着手机的你对他有点陌生，但在IT行业，他的名字就是硅谷精神的象征。1936年出生的格鲁夫，从一个匈牙利难民一路跌爬滚打成为芯片巨头英特尔的奠基者和精神领袖，他对整个科技行业都有非凡的意义。有人如此形容他：格鲁夫领导了整个PC时代，然后把它带走了。 格鲁夫出生于一个布达佩斯的犹太人家庭，童年生活就是一个大写的“难”字。他4岁的时候，布达佩斯流行猩红热，他不幸感染，虽然后来捡回了一条命，但是听力却终生受损。8岁那年，匈牙利被德国纳粹占领，格鲁夫的50万犹太人同胞都被关进了纳粹集中营。格鲁夫和母亲凭借假身份躲藏在朋友家中，而父亲则被抓从事苦力，直到二战结束后才与家人团聚。 20岁是格鲁夫的人生转折点。这一年，匈牙利爆发十月革命，苏联坦克开进布达佩斯，他坐船偷渡出境，一路流亡到美国。在格鲁夫的回忆录《游向彼岸》中，他这样描述自己20岁前的那段岁月：我相继经历了匈牙利政府的法西斯独裁统治、德军占领匈牙利、纳粹的“最终解决”、苏联红军包围布达佩斯、“二战”刚刚结束之后的民主混乱时期、匈牙利共产党执政，以及被镇压在枪口下的民众起义等时期和事件。当时，约有20万匈牙利人逃到西方——我便是他们中的一个。 只有偏执狂才能生存 1956年，格鲁夫踏上了美国的国土。一开始，他基本不会英语，说话带着浓重的匈牙利口音。不过他努力克服了这一障碍，顺利进入了纽约州立大学并以几乎全A的成绩毕业，而后进入了加州伯克利，获得了博士学位。1963年，格鲁夫进入硅谷知名半导体公司仙童。大家可能对这家公司很陌生，那么小M就来跟大家解释一下它在全球科技发展史上的意义：仙童公司和硅谷崛起紧密相关，硅谷之后有无数闪闪发光的初创企业，他们的技术和管理人才都是从仙童出去的。总之你就这样理解吧，仙童是全球电子、半导体和电脑行业的“西点军校”。在仙童待的这几年，让格鲁夫成了集成电路领域的专家。 1968年，格鲁夫离职仙童加入了英特尔，从工程总监开始，一步步走上首席运营官、总裁、CEO的位置，最后成为了董事长。大家都知道，英特尔的处理器在业内一家独大、无人能敌，然而最早的时候，它其实是靠着记忆体打开的市场。而在业务转型的过程中，格鲁夫扮演了重要的角色。1987年到1997年，格鲁夫担任英特尔CEO的这十年间，英特尔每年返还给投资者的回报率高达44%！看到这里，小M整个人都惊了！
